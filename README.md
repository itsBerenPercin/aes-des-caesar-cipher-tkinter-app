# aes-des-caesar-cipher-tkinter-app
Description

Tkinter GUI app implementing AES, DES, Caesar

Encrypt/decrypt messaging

Integrity verification for AES/DES

Educational purpose

Features

AES-CBC with HMAC verification

DES with checksum validation

Caesar cipher

Dark/light theme switch

Key length enforcement

Usage

Choose algorithm

Enter message

Enter key

Encrypt / Decrypt

Requirements

Python 3.x

pycryptodome

tkinter (built-in)

Security Notes

DES & Caesar are insecure by design

AES uses integrity check to prevent wrong-key output

Not production-grade
